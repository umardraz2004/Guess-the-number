Number Guessing Game Instructions

Welcome to the Number Guessing Game! Here's how to play:

    1. Objective
  • The goal of the game is to guess the randomly generated number between 1 and 100.
    2. How to play:
  • Enter a number between 1 and 100 in the input field.
  • Click the "Roll" button to submit your guess.
    3. Feedback
  • After each guess, you will receive feedback:
  • If your guess is too high, you'll see a message indicating it.
  • If your guess is too low, you'll see a message indicating it.
  • If your guess is correct, you'll receive a congratulatory message.
    4. New Number Generation:
  • Every time you make a guess and receive feedback, the game generates a new random number between 1 and 100 + 25 for the next round and so on.
    5. Clearing the Screen:
  • You can clear the result area by clicking the "Clear" button.
    6. Rules:
  • You can only submit one guess at a time.
  • Please wait for the feedback message to finish typing before making another guess.
  • There are 5 level of game, In each level there will be a increment of 25 numbers,

    So that as you move along the random number will become tough to guess.
    7. Winning:
  • Keep guessing until you correctly guess the number.
  • Once you guess the correct number, you can start guessing the new number generated for the next round.
Enjoy the game and happy guessing!
